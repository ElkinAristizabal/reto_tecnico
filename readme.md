
# Playwright with Cucumber - API
El objetivo de este proyecto es darle cumplimiento al reto tecnico que fue suministrado previamente.

## Prerrequisitos
- Node.js
- Visual Studio Code
- Instalar las extensiones de:
    - Cucumber
    - Playwright

## Configuración del proyecto

1. Clonar el proyecto.
2. Ejecutar en la raiz del proyecto.
```bash
npm install
```
Para instalar dependencias (Se pueden observar en el package.json)
 
3. Ejecutar en la raiz del proyecto.

```bash
npm run test:qa
```

Para ejecutar el escenario de CRUD que se solicitó.

4. Ejecutar en la raiz del proyecto.

```bash
npm run report
```

Para visualizar el reporte del escenario.


Nota: Si bien es cierto se sube el archivo .env, esto solo se hace para fines practicos en ambientes productivos o reales no sería buena practica.
